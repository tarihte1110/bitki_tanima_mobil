# Bitki Tanıma Mobil

86 farklı bitki türünü cihaz üzerinde (offline) tanıyabilen, React Native + Expo tabanlı mobil uygulama. Görsel seçildikten sonra önce “bitki var mı?” doğrulaması yapılır, ardından tür tahmini gösterilir.

## Özellikler
- Offline inference (TensorFlow.js)
- Kamera ve galeri desteği
- Top‑K tahmin listesi ve güven skoru
- Bitki detayları (zehirlilik, yenilebilirlik, açıklama, coğrafya)
- FloraMentor: yapay zeka botanik asistanı

## Kullanılan Teknolojiler
- React Native + Expo
- TensorFlow.js (React Native)
- MobileNet (bitki var mı ön filtresi)
- React Navigation
- Expo Image Picker / File System / Sharing
